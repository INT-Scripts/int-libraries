# INT Libraries 🏛️

A collection of asynchronous Python libraries for interacting with IMT-BS and Télécom SudParis student services.

## Packages

| Package | Description |
| :--- | :--- |
| **cas-connector** | Unified async CAS authentication |
| **trombint** | Student directory scraper |
| **si-agenda** | Agenda/Timetable synchronization |
| **tsprint** | PaperCut printing integration |
| **laver-int** | Campus laundry tracking |

## Development

This project uses `uv` workspaces. To install all packages in editable mode:

```bash
uv sync
```
